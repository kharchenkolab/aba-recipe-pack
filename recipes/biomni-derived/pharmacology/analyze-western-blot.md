# Analyze Western Blot

Distilled from a biomni implementation. In ABA, implement with the tools below — not biomni.

## Approach
1. Load the blot image with `skimage.io.imread`; convert to grayscale by averaging channels if RGB.
2. Extract loading-control ROI: `image[y:y+h, x:x+w]`; compute `lc_intensity = np.sum(roi)`.
3. For each target band ROI, compute `band_intensity = np.sum(roi)`.
4. Compute relative expression: `band_intensity / lc_intensity`.
5. Write results to a CSV: protein name, raw intensity, relative expression.
6. Return a formatted research log summarizing antibody info, steps, and results.

## Key decisions
- ROIs are provided as `[x, y, width, height]` and can be generated by `find_roi_from_image`.
- Normalization is a simple ratio to loading control; no background subtraction is applied.
- Sum intensity (not mean) is used; scales with band area, which is appropriate for standard blot densitometry.

## Caveats
- Background subtraction (rolling-ball or lane-adjacent background) is not performed; results may be skewed on high-background blots.
- Grayscale conversion by channel-mean may underweight fluorescent-channel specificity in multiplexed blots.
